**Escape From Alex** is a game made with Unity and C#. It is a 3D  game Horror where you have to escape from Alex.

**Synopsis: **
find exit and escape from Alex

**Gameplay:**
- You have to escape from Alex
- You have to find exit

**Controls:**
- WASD to move
- Space to jump
